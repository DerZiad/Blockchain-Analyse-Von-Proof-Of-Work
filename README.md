# Blockchain Technologies — Analyse von Proof of Work

Kurzbeschreibung (DE)
Dieses Repository enthält eine Seminararbeit zur Analyse von Proof of Work (PoW) sowie Begleitmaterialien und ein kleines Python-Beispiel zur Veranschaulichung der Mining-Idee.

Brief description (EN)
This repository holds a seminar report analyzing Proof of Work (PoW) blockchain concepts, accompanying presentation slides, and a small Python example demonstrating mining logic.

Contents
- Full report (PDF): datein/Seminar.pdf
- Example code (Python): datein/mine.py
- Presentation (PPTX): datein/SeminarPräsentation.pptx

Quick start / Kurzanleitung
- View the report: open `datein/Seminar.pdf`.
- Run the example miner (requires Python 3.8+):
  - cd into the repository root
  - python datein/mine.py
- No external dependencies are required for the simple example, but inspect the file header for details.
